# How to generate

The design assumes printing on neon green paper, if you want to be fancy it's available as UV fluorescent paper.

Settings:
```bash
../../generate_ecash_pdf.py --front-image ../../contrib/39c3/front.png --back-image ../../contrib/39c3/back.png --qr-size 4.2 --qr-x-offset 0.9 --qr-y-offset 0.87 --qr-error-correction Q notes.csv
```
